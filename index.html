MeshiCam プライバシーポリシー

最終更新日：2026年3月4日

本プライバシーポリシーは、MeshiCam（以下「本アプリ」）が収集する情報、その利用目的、第三者への提供、およびユーザーの権利について定めるものです。本アプリをご利用いただくことで、本ポリシーに同意いただいたものとみなします。

1. 収集する情報

本アプリでは、サービスの提供に必要な範囲で以下の情報を収集します。

データの種類 内容 収集タイミング
認証情報 Firebase Authentication を通じて取得するユーザーID、メールアドレス、表示名。匿名認証の場合は匿名IDのみ。 アカウント作成・ログイン時
食材写真 ユーザーがカメラで撮影または端末から選択した食材の画像データ 食材撮影・画像選択時
食材データ AI が認識した食材名、カテゴリ、数量 食材認識完了時
レシピ履歴 提案されたレシピの名前、材料、手順、調理時間等 レシピ提案時
調理記録 実際に調理を完了したレシピの記録（レシピ名、調理日時、カロリー推定値等） 調理完了時
利用回数データ レシピ提案機能の利用日時 レシピ提案リクエスト時

本アプリは、位置情報、連絡先、通話履歴など上記以外の個人情報は収集しません。

2. データの利用目的

収集した情報は、以下の目的でのみ利用します。

  • レシピ提案の提供：登録された食材情報をもとに、AI がレシピを生成・提案します。
  • レシピ提案の精度向上：ユーザーの食材データや調理履歴を分析し、より適切なレシピ提案を行います。
  • アプリの改善：利用状況の統計的分析を通じて、機能改善やユーザー体験の向上に役立てます。
  • 無料プラン管理：利用回数データを用いて、週ごとの無料利用回数を管理します。
  • アカウント管理：認証情報を用いて、ユーザーごとのデータ管理およびログイン機能を提供します。

3. 第三者への情報提供

本アプリは、以下のサービスとデータを共有します。

3.1 Anthropic（Claude API）

食材認識およびレシピ生成のため、ユーザーが撮影した食材画像および食材テキストデータを Anthropic 社の Claude API に送信します。送信されたデータは Anthropic 社のプライバシーポリシーに基づいて処理されます。Anthropic 社は、API 経由で送信されたデータをモデルのトレーニングに使用しません。

  • 送信データ：食材画像（Base64 エンコード）、食材リスト、調理条件
  • 利用目的：食材認識、レシピ生成
  • Anthropic プライバシーポリシー：https://www.anthropic.com/privacy

3.2 Google / Firebase

本アプリはバックエンドインフラとして Google Firebase を利用しています。以下のサービスを通じて、Google のインフラストラクチャ上でデータが処理・保存されます。

  • Firebase Authentication：ユーザー認証の管理
  • Cloud Firestore：食材データ、調理記録、利用回数等の保存

Google のプライバシーポリシー：https://policies.google.com/privacy

3.3 その他

上記以外の第三者に対して、ユーザーの個人情報を販売・提供することはありません。ただし、法令に基づく開示要求があった場合はこの限りではありません。

4. データの保存場所と保護

4.1 保存場所

ユーザーデータは、Google Cloud Firestore の東京リージョン（asia-northeast1）に保存されます。データは Google のセキュリティ基準に基づき、転送中および保存時に暗号化されます。

4.2 保存期間

  • 認証情報：アカウント削除まで保持
  • 食材データ：ユーザーが削除するか、アカウント削除まで保持
  • 調理記録：アカウント削除まで保持
  • 利用回数データ：アカウント削除まで保持
  • 食材画像：AI 処理完了後、サーバー上には保存しません。端末のメモリ上でのみ一時的に使用されます。

4.3 セキュリティ対策

  • Firebase Security Rules によるアクセス制御（ユーザーは自身のデータのみアクセス可能）
  • 通信の TLS / SSL 暗号化
  • API キーの安全な管理

5. ユーザーの権利

ユーザーは自身の個人情報に関して、以下の権利を有します。

5.1 データの確認

アプリ内の「食材」画面、「履歴」画面、「設定」画面から、保存されているデータを確認できます。

5.2 データの削除

  • 個別データの削除：食材データはアプリ内から個別に削除できます。
  • アカウントの削除：アカウント削除をご希望の場合は、下記のお問い合わせ先までご連絡ください。アカウントに紐づくすべてのデータ（認証情報、食材データ、調理記録、利用回数データ）を削除いたします。

データ削除のご依頼には、原則として30日以内に対応いたします。

5.3 同意の撤回

本ポリシーへの同意は、アプリのアンインストールおよびアカウント削除によりいつでも撤回できます。

6. 子どものプライバシー

本アプリは13歳未満のお子様を対象としておりません。13歳未満のお子様から意図的に個人情報を収集することはありません。万が一、13歳未満のお子様の情報が収集されたことが判明した場合は、速やかに削除いたします。

7. プライバシーポリシーの変更

本ポリシーは、法令の改正やサービス内容の変更に伴い、事前の通知なく更新される場合があります。重要な変更がある場合は、アプリ内通知またはその他の適切な方法でお知らせします。最新のポリシーは常にこのページでご確認いただけます。
